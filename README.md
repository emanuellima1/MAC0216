# MAC0216

## Aventura!

Para rodar o jogo:

```
make
./jogo.out
```


Para rodar testes:

```
$ make test
```

Em seguida, para testar as bibliotecas de lista, tabela, e elemento:

```
$ ./lib-test
```

E para testar o funcionamento básico do jogo:

```
$ ./jogo-test
```



Para apagar arquivos temporários:

```
$ make clean
```
